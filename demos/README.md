# Few-shot prompts of FoVer

The files in this folder are structured using **Jupyter-style cells** (marked with `# %%`).

The system prompt is included in the first runnable cell, indicated with `# %% System prompt:`.  
The subsequent runnable cells contain the few-shot examples,
each including a User-Assistant pair
(marked with `## User:` and `## Assistant:` respectively).
